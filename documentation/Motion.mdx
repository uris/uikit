            # Slice Motion
            Slice is a product-first UI kit. Motion and state transitions help the
            interface feel natural, responsive, and efficient.

            Slice has three core transition presets for different needs and situations:

            #### "Water"
            Water is a smoother, more natural alternative to a standard ease-in-out
            curve. It works well for most UI state transitions like hover state colors.
            This is the main transition used by Slice like hover colors, etc.
            <MotionSimulation easing={"water"} duration={0.25} />
            <Spacer size={24} />

            #### "Magnet"
            Magnet is an ease-out curve with a "snap into place" feel. It's used for
            elements that move over longer distances, such as panels and drawers. It's default duration is longer than water, but it feels "snappier".
            <MotionSimulation easing={"magnet"} duration={0.5} />
            <Spacer size={24} />

            #### "Spring"
            Spring has a "bounce" effect and is used sparingly - mostly for emphasis. An example is the [done checkmark](/story/components-donecheck--default). 
            <MotionSimulation easing={"spring"} duration={1} />

            Note: Spring uses a scale value so the smaller the difference in the from/to animated values, the less exaggerated the effect will seem.
            <MotionSimulation placeholder={"Short spring"} width={250} easing={"spring"} duration={1} />
            <Spacer size={24} />
