        # useMDStreamBuffer
        React hook wrapper around `MdBuffer`. The `MdBuffer` utility transforms partial markdown streams into "healthy" markdown ready for rendering without re-render flickers. The hook manages the buffer lifecycle for you and exposes the current healthy markdown output plus the core buffer actions.
            
        ## Not a Parser
        `MdBuffer` is not a markdown to react parser. It simply feeds a parser like [react-marked](https://www.npmjs.com/package/react-markdown) or [marked](https://www.npmjs.com/package/marked) "healthy" markdown for efficient parsing and rendering. See [MdBuffer](/docs/utils-markdown-buffer-class--docs) class for more details.
            
        ## Signature
        `useMDStreamBuffer(options?)`

        ## Basic example
        </Unstyled>

        <RenderSource language={"tsx"}>
        ```
        import { useEffect } from 'react';
        import { useMDStreamBuffer } from '@apple-pie/slice';

        function StreamingMarkdown({ chunk }: { chunk: string }) {
            const { healthy, append } = useMDStreamBuffer({
                includeLinksAndImages: true,
            });

            useEffect(() => {
                if (!chunk) return;
                append(chunk);
            }, [chunk, append]);

            return <div>{healthy}</div>;
        }
        ```
