        # useMDStreamBuffer
        React hook wrapper around `MdBuffer`. The `MdBuffer` utility transforms partial markdown streams into "healthy" markdown ready for rendering without re-render flickers and trailing markdown tokens.

        The hook also supports optional upstream pacing. This lets you intentionally slow very fast incoming streams, or break up large chunk streams into smaller updates.
            
        ## Not a Parser!
        `MdBuffer` is not a markdown to react parser - is is more of a pre-processor. It simply feeds a parser like [react-marked](https://www.npmjs.com/package/react-markdown) or [marked](https://www.npmjs.com/package/marked) "healthy" markdown for efficient parsing and rendering. See [MdBuffer](/docs/utils-markdown-buffer-class--docs) class for more details.
            
        ## Signature
        `useMDStreamBuffer(options?)`

        #### End Marker
        End markers can be uses to display animated progress cursors. `options.healthyEndMarker` appends an ephemeral suffix to `healthy` output without changing `raw`. If the buffer has optimistically closed a line-based construct by adding a trailing newline, the marker is inserted before that newline.

        ## Basic example
        </Unstyled>

        <RenderSource language={"tsx"}>
        ```
        import { useEffect } from 'react';
        import { useMDStreamBuffer } from '@apple-pie/slice';

        function StreamingMarkdown({ chunk }: { chunk: string }) {
            const { healthy, append } = useMDStreamBuffer({
                includeLinksAndImages: true,
                paceDelayMs: 40,
                paceChunkSize: 12,
            });

            useEffect(() => {
                if (!chunk) return;
                append(chunk);
            }, [chunk, append]);

            return <div>{healthy}</div>;
        }
        ```
        </RenderSource>

        <Unstyled>
        In the example above, incoming chunks are first queued by the hook and then released into `MdBuffer` every `40ms`, up to `12` characters at a time.
