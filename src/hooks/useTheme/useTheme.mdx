        # useTheme
        Main Slice theming hook. Returns the active theme plus helpers and token maps for CSS variables, class names, and typography.

        ## Signature
        `useTheme()`

        ## Parameters
        This hook does not take parameters.

        ## Returns
        <table style={{width: '100%'}}>
          <colgroup>
            <col style={{width: '1%'}} />
            <col style={{width: '1%'}} />
            <col style={{width: '1%'}} />

          </colgroup>

            <tr><td><code>darkTheme</code></td><td><code>SliceTheme</code></td><td><code>-</code></td><td>Default Slice dark theme object.</td></tr>
            <tr><td><code>lightTheme</code></td><td><code>SliceTheme</code></td><td><code>-</code></td><td>Default Slice light theme object.</td></tr>
            <tr><td><code>current</code></td><td><code>SliceTheme</code></td><td><code>-</code></td><td>Currently active theme.</td></tr>
            <tr><td><code>colors</code></td><td><code>Record&lt;string, string&gt;</code></td><td><code>colorCssVars</code></td><td>Color tokens mapped to CSS vars (for example <code>var(--core-text-primary)</code>).</td></tr>

        ## Code Examples
        </Unstyled>

        <RenderSource language={"tsx"}>
        ```
        import { useTheme } from '@apple-pie/slice';
        
        function ThemeToggle() {
            const theme = useTheme();
        
            return (
                <button
                    type="button"
                    className={theme.colorsClass['core-text-special']}
                    onClick={() => theme.toggle()}
                >
                    Current: {theme.current.name}
                </button>
            );
        }
        ```
