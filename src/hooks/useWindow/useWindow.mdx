        # useWindow
        Tracks viewport size, form factor, touch capability, pixel ratio, and Electron runtime state.

        ## Signature
        `useWindow(top?: boolean)`

        ## Parameters
        <table style={{width: '100%'}}>
            <colgroup>

            <thead>
            <tr>
                <th>Parameter</th>
                <th>Type</th>
                <th>Default</th>

            <tbody>
            <tr><td><code>Top?</code></td><td><code>boolean</code></td><td><code>true</code></td><td>Use to top most frame to set width and height of the viewport. If false, uses the current frame.</td></tr>
