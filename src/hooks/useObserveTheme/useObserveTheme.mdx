        # useObserveTheme
        Observes <code>document.documentElement.dataset.theme</code> and returns the resolved Slice theme object.

        ## Signature
        `useObserveTheme()`

        ## Parameters
        This hook does not take parameters.

        ## Returns
        <table style={{width: '100%'}}>
          <colgroup>

          </colgroup>

              <th>Return</th>
              <th>Type</th>

          </tbody>
        </table>

        </Unstyled>
