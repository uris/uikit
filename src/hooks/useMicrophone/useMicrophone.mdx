        # useMicrophone
        Requests microphone access, keeps both the raw microphone stream and a processed microphone stream in refs, and exposes controls for muting, input gain, switching devices, refreshing available devices, and stopping the stream.

        ## Behavior
        By default, the hook requests microphone access on mount. Pass `autoRequest={false}` to delay permission prompting and call `requestMicrophone()` manually when you are ready. It also listens for `navigator.mediaDevices` `devicechange` events, refreshes the available microphone list when devices come online or disconnect, and falls back to the default device if the currently selected microphone disappears.

        ## Signature
        `useMicrophone(startMuted?, microphoneDeviceId?, autoRequest?)`

        ## Notes
        - `micStream`, `processedMicStream`, and `micTrack` are refs on purpose. They hold live media objects and do not force rerenders when the underlying stream changes.
        - `inputVolume` controls app-side gain through Web Audio. It does not change the system or hardware microphone gain.
        - If a preferred device cannot be found during a request, the hook retries against the browser's default microphone.
        - If the active microphone is unplugged or removed, the hook refreshes devices and reconnects to the default input.
        - When `autoRequest` is `false`, the hook still refreshes the device list on mount without prompting for microphone access.
        - Chrome may expose placeholder `audioinput` entries before permission is granted. The hook now suppresses those unlabeled placeholders and refreshes the list again immediately after a successful connection so `micOptions` stays aligned with the real devices.

        ## Basic example
        </Unstyled>

        <RenderSource language={"tsx"}>
        ```
        import { useMicrophone } from '@apple-pie/slice';

        function MicrophoneStatus() {
            const { isActive, muted, toggleMute, micTrack, micOptions, isRequesting } = useMicrophone(false);

            return (
                <div>
                    <div>Requesting: {String(isRequesting)}</div>
                    <div>Active: {String(isActive)}</div>
                    <div>Track ready: {String(micTrack.current?.readyState === 'live')}</div>
                    <div>Available mics: {micOptions.length}</div>
                    <div>Muted: {String(muted)}</div>
                    <button type="button" onClick={() => toggleMute()}>
                        Toggle Mic
                    </button>
                </div>
            );
        }
        ```
        </RenderSource>

        <RenderSource language={"tsx"}>
        ```
        import { useMicrophone } from '@apple-pie/slice';

        function ManualMicrophoneRequest() {
            const { requestMicrophone, micTrack, isRequesting } = useMicrophone(
                true,
                undefined,
                false,
            );

            return (
                <div>
                    <div>Track ready: {String(micTrack.current?.readyState === 'live')}</div>
                    <button
                        type="button"
                        disabled={isRequesting}
                        onClick={() => void requestMicrophone()}
                    >
                        Request Microphone
                    </button>
                </div>
            );
        }
        ```
