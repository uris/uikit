        # useMicrophone
        Requests microphone access, keeps a live microphone stream in refs, and exposes simple controls for muting, switching devices, refreshing available devices, and stopping the stream.

        ## Behavior
        The hook requests microphone access on mount. It also listens for `navigator.mediaDevices` `devicechange` events and refreshes the available microphone list when devices come online or disconnect.

        ## Signature
        `useMicrophone(startMuted?, microphoneDeviceId?)`

        ## Basic example
        </Unstyled>

        <RenderSource language={"tsx"}>
        ```
        import { useMicrophone } from '@apple-pie/slice';

        function MicrophoneStatus() {
            const { muted, toggleMute, micTrack, microphones } = useMicrophone(true);

            return (
                <div>
                    <div>Track ready: {String(micTrack.current?.readyState === 'live')}</div>
                    <div>Available mics: {microphones.length}</div>
                    <div>Muted: {String(muted)}</div>
                    <button type="button" onClick={() => toggleMute()}>
                        Toggle Mic
                    </button>
                </div>
            );
        }
        ```
