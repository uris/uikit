        # ThemeProvider
        Provides Slice theme behavior by writing `data-slice-theme` on `document.documentElement`, rendering a scoped wrapper with `data-slice-theme-scope`, and loading Slice CSS token/type styles.
        When the provider resolves an explicit or initial theme on the client, it also persists a `slice-theme` cookie so SSR helpers can read the same value on the server for the next request.

        ## Base element
        `div`

        ## Interface
        `ThemeProviderProps`
