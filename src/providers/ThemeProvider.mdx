        # ThemeProvider
        Provides Slice theme context behavior by writing `data-theme` on `document.documentElement` and loading Slice CSS token/type styles.

        ## Base element
        None (provider component, no wrapper element)

        ## Interface
        `ThemeProviderProps`

        ## Props
