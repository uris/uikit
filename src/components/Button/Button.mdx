        # Button
        Renders a versatile button supporting variants, icons, loading/progress, and interaction states.

        Use `size="small"` for the compact button layout. `text` is a `variant`, not a `size`.

        ## Base element
        `HTMLButtonElement`

        ## Interface
        `ButtonProps`
