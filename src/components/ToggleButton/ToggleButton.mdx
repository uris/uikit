        # ToggleButton
        A small composition on top of `IconButton` that pairs an icon toggle with an adjacent label.
        Use it for simple selected/unselected controls or grouped settings toggles where the icon remains the primary visual affordance.

        ## Behavior
        - `ToggleButton` is controlled through `selected`.
        - Clicking toggles the next state and calls `onChange(nextState)`.
        - `onSelect()` only fires when the next state becomes `true`.
        - When `unselect={false}`, an already-selected toggle cannot be turned off by clicking it again.
        - Visual button behavior such as sizing, hover states, and tooltip rendering comes from the underlying `IconButton`.

        ## Code Example
        </Unstyled>

        <RenderSource language={"tsx"}>
        ```
        import { useState } from 'react';
        import { ToggleButton } from '@apple-pie/slice';

        export function ThemeToggleExample() {
            const [selected, setSelected] = useState(false);

            return (
                <ToggleButton
                    label="System"
                    icon="keyboard"
                    selected={selected}
                    unselect={false}
                    onChange={setSelected}
                />
            );
        }
        ```
