        # Switch
        Renders a binary on/off control for toggling settings and preferences.

        ## Base element
        `HTMLButtonElement`

        ## Interface
        `SwitchProps<T = string>`

        ## Typing
        `Switch` is generic. The `value` prop uses the same `T` that comes back through `onChange`.

        `onChange` receives:

        `(value?: T, state: boolean) => void`

        ## Notes
        - `Switch<string>` is the default if you do not provide a generic.
        - Use `Switch<T>` when the switch should carry a typed value object or non-string identifier.
        - `state` controls the visual on/off state, and `value` is the optional typed payload returned in `onChange`.
