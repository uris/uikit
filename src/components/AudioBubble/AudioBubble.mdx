        # AudioBubble
        Renders a simple circular bubble that reacts to live audio using the `AudioVisualizer` utility.
        It is meant for speaking indicators, voice orbs, or other UI that should react to audio activity.

        ## Base element
        `HTMLDivElement`

        ## Interface
        `AudioBubbleProps`

        ## Props

        ## Code Example
        </Unstyled>

        <RenderSource language={"tsx"}>
```
import { AudioBubble } from '@apple-pie/slice';
import { useMicrophone } from '@apple-pie/slice/hooks';

export function AudioBubbleExample() {
    const { micStream } = useMicrophone(false);

    return (
        <AudioBubble
            audioStream={micStream.current}
            playing={true}
            size={72}
            backgroundColor="#ffffff"
            glowColor="rgba(255, 92, 61, 0.28)"
            minScale={1}
            maxScale={2}
            peakIntensity={0.2}
        />
    );
}
```
