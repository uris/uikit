        # Avatar
        Renders a user profile image or fallback initials. Use it for identity surfaces where an image may or may not be available.

        ## Base element
        `HTMLDivElement`

        ## Interface
        `React.HTMLAttributes<HTMLDivElement> & AvatarProps`
