        ## Notes
        - The component auto-resizes to fit its content as the user types.
        - Focus state takes precedence over `error` for border color.
        - When `returnSubmits` is enabled, pressing Enter triggers `onSubmit` and prevents a newline.
        - When `submitClears` is enabled, the local textarea state is reset after submit.
