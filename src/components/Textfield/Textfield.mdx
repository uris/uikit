        # TextField
        Renders a single-line text input with optional label, validation, clear control, and password visibility toggle.

        ## Base element
        `HTMLDivElement`

        The component renders a styled wrapper around a native `input`.

        ## Interface
        `TextFieldProps`

        ## Notes
        - Pressing Enter triggers <code>onSubmit</code>, then blurs the input.
        - The clear control only renders its icon while the field is focused and has content.
        - In password mode, the visibility toggle is shown unless <code>noShow</code> is true.
        - Use the <code>error</code> prop to render the error colors; the component no longer performs a built-in validation pass.
        - The style API is flattened. Use the state-specific props directly instead of grouped objects.
