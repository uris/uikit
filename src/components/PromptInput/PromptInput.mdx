        # PromptInput
        Renders a flexible, auto-resizing prompt textarea for chat and AI message boxes, with working/stop states, configurable border styles, optional attach/send controls, and customizable toolbar content.
        Choose border style of gradient to apply an "ai" gradient affordance to the border.
        Attachment chips are controlled via `attachments` + `onAttachmentsChange` (parent owns attachment state).

        ## Base element
        `HTMLDivElement`

        ## Interface
        `PromptProps`

        ## Code Examples
        </Unstyled>

        <RenderSource language={"tsx"}>
{`
import { useState } from 'react';
import { PromptInput } from '@apple-pie/slice/components/PromptInput/PromptInput';

export function PromptInputExample() {
  const [attachments, setAttachments] = useState([
    { file: 'policy.pdf' },
    { file: 'notes.txt' },
  ]);

  return (
    <PromptInput
      attachments={attachments}
      value="Summarize this policy update"
      placeholder="Ask the assistant anything ..."
      borderStyle="gradient"
      textSize="m"
      onAttachmentsChange={(items) => setAttachments(items)}
      onSubmit={(value) => console.log(value)}
      id="promptinput-example"
      data-testid="promptinput-example"
      className="custom-wrapper"
      style={{ marginTop: 8, maxWidth: 720 }}
    />
  );
}
`}
        </RenderSource>

        ## Custom Toolbar Tools
        JSX you write inside the prompt tags will render as `children` appearing as custom tools into the bottom toolbar area next to the attach button (you can toggle attach on/off). This is useful for context controls like model selection, tone, or output format.

export function PromptInputWithModelSelector() {

  return (
    <PromptInput
      placeholder="Ask a question..."
      onSubmit={(value) => console.log({ value, model })}

          onChange={(e) => setModel(e.target.value)}
          style={{ height: 28 }}

          <option value="gpt-4.1">GPT-4.1</option>
          <option value="gpt-4o-mini">GPT-4o mini</option>
          <option value="gpt-4.1-mini">GPT-4.1 mini</option>
        </select>

  );
}
