        # Level
        Renders a horizontal audio level meter driven by the `AudioVisualizer` utility.
        It maps normalized microphone or stream intensity into a row of bars, then lights up bars from left to right as the signal gets stronger.

        ## Base element
        `HTMLDivElement`

        ## Interface

        ## Behavior
        - `maxIntensity` controls the total number of bars.
        - `minIntensity` controls the resting number of active bars.
        - Live audio intensity is normalized to `0..1`, then mapped into an active bar count between `minIntensity` and `maxIntensity`.

        ## Code Example
        </Unstyled>

        <RenderSource language={"tsx"}>
```
import { Level } from '@apple-pie/slice';
import { useMicrophone } from '@apple-pie/slice/hooks';

export function AudioLevelExample() {
    const { micStream } = useMicrophone(false);

    return (
        <Level
            audioStream={micStream.current}
            playing={true}
            width={160}
            height={6}
            gap={4}
            maxIntensity={8}
            minIntensity={0}
            colorActive="var(--feedback-positive)"
            backgroundColor="var(--core-surface-primary-tint)"
        />
    );
}
```
