        ## TabOption
        `TabOption` supports:

        `name?: string`

        `value?: any`

        `icon?: string | null`

        `toolTip?: string`

        `count?: number`

        `component?: any`

        ## Notes
        - `selectedValue` takes precedence when it matches an option value. Otherwise the component falls back to `selected`.
        - `tabWidth="fill"` makes tabs flex across the available space.
        - `tabWidth="compact"` makes each tab size to its content.
        - `tabWidth={number}` applies a fixed pixel width to each tab.
