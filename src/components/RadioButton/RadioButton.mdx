        # RadioButton
        Renders a single radio or checkbox-style option with typed value support.

        ## Base element
        `HTMLButtonElement`

        ## Interface
        `RadioButtonProps<T = string>`

        ## Related types
        `RadioButtonOption<T = string>`

        ## Typing
        `RadioButton` is generic. The `value` prop uses the same `T` that comes back through `onChange`.

        `onChange` receives that same typed value:

        `(option: RadioButtonOption<T>, selected: boolean) => void`

        ## Notes
        - The component is generic, so `value` and `onChange` stay typed when you use `RadioButton<T>`.
        - `icon` is the unchecked icon and `checkedIcon` is the selected icon.

```
import { RadioButton } from '@apple-pie/slice';

export function RadioButtonExample() {
    return (
        <RadioButton
            label="Email alerts"
            fieldName="alerts"
            value="email-alerts"
            selected={true}
            onChange={(option, selected) => console.log(option, selected)}
            id="radiobutton-example"
            data-testid="radiobutton-example"
            className="custom-wrapper"
            style={{ marginTop: 8, maxWidth: 480 }}
        />
    );
}
```
