        # ErrorSummary
        `ErrorSummary` renders a grouped summary of errors with entry titles and one or more message lines per entry.

        ## Base element
        `HTMLDivElement`

        ## Interface
        `ErrorSummaryProps`

        ## Typing
        Error entries have the following shape:
        <RenderSource>
        ```
        type ErrorMessage = {
            title: string;
            message?: string | string[];
        };
        ```
        </RenderSource>

        ## Props

          <colgroup>

          <tbody>
            <tr><td><code>entries</code></td><td><code>ErrorMessage[]</code></td><td><code>undefined</code></td><td>List of grouped error entries to render. When omitted or empty, nothing is rendered.</td></tr>
            <tr><td><code>textColor</code></td><td><code>string</code></td><td><code>'var(--feedback-warning)'</code></td><td>Text color applied to the summary content through CSS variables.</td></tr>

        ## Events
        `ErrorSummary` does not expose component-specific callback props.

        ## Rendering behavior
        - The component only renders when `entries` exists and contains at least one item.
        - Each entry renders its `title` in bold.
        - If `message` exists, it is normalized into a list and rendered as bullet items.
        - When `autoNumber` is `true` and there is more than one entry, titles are prefixed with their item number.
        - `textSize`, `textColor`, and `backgroundColor` are applied through the component CSS variables.
        - The summary uses motion-based enter/exit animation when it appears or disappears.

        ## Code examples
        </Unstyled>

        <RenderSource>
        ```
        import { ErrorSummary } from '@apple-pie/slice';
            
        // error list
        const errors = [
            { title: 'Email is required' },
            {
                title: 'Invalid Password Format',
                message: [
                    'Password must be at least 8 characters',
                    'Password must include a number',
                ],
            },
        ]
            
        export function ErrorSummaryExample() {
            return (
                <ErrorSummary autoNumber={true} entries={errors} />
            );
        }
        ```
