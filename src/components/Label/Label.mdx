        # Label
        Renders a compact text label for statuses, tags, and lightweight inline actions.

        ## Base element
        `HTMLSpanElement | HTMLButtonElement`

        If `onClick` is provided, `Label` renders a `button`. Otherwise it renders a `span`.

        ## LabelBackground
        `LabelBackground` includes the semantic values:

        `red | yellow | green | blue | grey | lightGrey | white`

        ## Notes
        - Semantic `bgColor` values map both the background and border color to theme tokens.
        - Passing a custom CSS color string to `bgColor` uses that value directly as the background.
        - `children` overrides `label` when both are provided.
