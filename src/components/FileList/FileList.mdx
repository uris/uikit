        # FileList
        Renders a list of files as removable chips with file-type icons. It supports upload-aware states per item, including in-progress uploads (`uploading` + `progress`) and error messaging (`error`).

        `FileList` is a controlled component: clicking remove emits `onChange(updatedFiles)` but does not mutate internal list state. The parent must pass the updated `files` prop for the UI to change.

        ## Base element
        `HTMLDivElement`
