        # TipStore
        Global tooltip state helpers built with Zustand. Supports atomic selector hooks for React and imperative access outside React.

        ## React exports (reactive)
        React exports are reactive subscriptions for components updating automatically as state changes/mutates.

        ## Imperative exports (non-reactive)
        These APIs call the same singleton store instance directly. Use them for service layers, event handlers, and utilities where hooks are not allowed. Imperative exports are direct store reads/writes and do not trigger component subscription behavior.

        ## Atomic selectors in React
        Use focused selectors so components only re-render when their selected slice changes.

        <RenderSource language={"tsx"}>
{`
import { useTip, useTipActions } from '@apple-pie/slice/stores/tip';

function TipLauncher() {
  const tip = useTip();
  const actions = useTipActions();

  return (
    <button
      type="button"
      onMouseEnter={(event) =>
        actions.push({ payload: { label: 'Go home' }, event })
      }
      onMouseLeave={() => actions.clear()}
    >
      {tip?.payload?.label ?? 'Hover me'}
    </button>
  );
}
`}
        </RenderSource>

        ## Generic selector usage in React
        Use `useTipStore` directly when you need a custom selector.

        <RenderSource language={"tsx"}>
{`
import { useTipStore } from '@apple-pie/slice/stores/tip';

function TipLabel() {
  const label = useTipStore((state) => state.tip?.payload?.label ?? null);
  return <span>{label ?? 'no tip'}</span>;
}
`}
        </RenderSource>

        ## Imperative usage outside React
        Use imperative helpers in utility modules, services, and non-component handlers.
        These are non-reactive reads/writes.

        <RenderSource language={"tsx"}>
{`
import { tipActions, getTip } from '@apple-pie/slice/stores/tip';

export function setGlobalTip(label: string) {
  tipActions.push({ payload: { label } });
}

export function clearGlobalTip() {
  tipActions.clear();
}

export function getCurrentTipLabel() {
  return getTip()?.payload?.label ?? null;
}
`}
