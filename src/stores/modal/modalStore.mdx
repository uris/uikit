        # ModalStore
        Global modal orchestration helpers built with Zustand. The store tracks one active modal at a time and supports both immediate show or hide flows and promise-based modal responses.

        ## React exports (reactive)
        React exports are reactive subscriptions for components updating automatically as state changes.

        ## Actions
        Use `show()` for fire-and-forget display and `modalResponse()` when the caller should await the result. Explicit button choices should resolve. Dismissal, closing, or clearing should reject.

        ## Atomic selectors in React
        Use focused selectors so components only re-render when their selected slice changes.

        <RenderSource language={"tsx"}>
        ```
        import { useModal, useModalActions } from '@apple-pie/slice/stores/modal';
        import { Button, Modal } from '@apple-pie/slice';
        
        function ModalLauncher() {
            const modal = useModal();
            const actions = useModalActions();
        
            return (
                <Button
                    onClick={() =>
                        actions.show({
                            component: Modal,
                            props: {
                                title: 'Confirmation',
                                children: 'Continue?',
                            },
                        })
                    }
                >
                    {modal ? 'Modal Open' : 'Open Modal'}
                </Button>
            );
        }
        ```
        </RenderSource>

        ## Promise-based modal response
        Use `modalResponse()` when callers should await the user's choice. Handle dismissal with `catch`.

        <RenderSource language={"tsx"}>
        ```
        import { useModalActions } from '@apple-pie/slice/stores/modal';
        import { Modal } from '@apple-pie/slice';
        
        function ConfirmButton() {
            const actions = useModalActions();
        
            const handleClick = async () => {
                try {
                    const result = await actions.modalResponse<string>({
                        component: Modal,
                        props: {
                            title: 'Delete item?',
                            children: 'This action cannot be undone.',
                            actions: [
                                { id: 'cancel', label: 'Cancel', value: 'cancel' },
                                { id: 'delete', label: 'Delete', value: 'delete', primary: true },
                            ],
                        },
                    });
            
                    console.log(result);
                } catch (error) {
                    console.error(error);
                }
            };
        
            return <button type="button" onClick={handleClick}>Delete</button>;
        }
        ```
        </RenderSource>

        ## Imperative usage outside React
        These APIs are useful for non-component event handlers and services.

        <RenderSource language={"tsx"}>
        ```
        import { Modal } from '@apple-pie/slice';
        import { modalActions, getModal } from '@apple-pie/slice/stores/modal';
        
        export function openSettingsModal() {
            modalActions.show({
                component: Modal,
                props: {
                    title: 'Settings',
                    children: 'Configure your preferences.',
                },
            });
        }
        
        export function dismissActiveModal() {
            if (getModal()) modalActions.clear();
        }
        ```
