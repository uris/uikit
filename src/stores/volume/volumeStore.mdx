        # VolumeStore
        Global media volume state built with Zustand. It tracks the active output volume, mute state, the last non-muted volume, a registry of attached audio/video elements that should be updated together, and an optional dedicated feedback element for short UI sounds.
        Volume values are normalized numbers between `0` and `1`.

        ## React exports (reactive)
        React exports are reactive subscriptions for components that should re-render when the selected slice changes.

            <tr><td><code>mute</code></td><td><code>{`() => Promise<void>`}</code></td><td>Sets output volume to zero while preserving the previous volume for restore, then plays the feedback element when present.</td></tr>

        ## Notes
        - `setVolume()` no longer plays the feedback sound by default. Existing callers that relied on automatic beeps must now pass <code>{`{ playFeedback: true }`}</code>.

        ## Example
        </Unstyled>

        <RenderSource language={"tsx"}>
        ```
        import { useEffect, useMemo } from 'react';
        import { Slider } from '@apple-pie/slice';
        import {
            useMuted,
            useStoredVolume,
            useVolume,
            useVolumeActions,
        } from '@apple-pie/slice/stores/volume';

        function AudioVolumePanel() {
            const volume = useVolume();
            const storedVolume = useStoredVolume();
            const muted = useMuted();
            const actions = useVolumeActions();
            const audioElement = useMemo(() => {
                if (typeof Audio === 'undefined') return null;
                return new Audio();
            }, []);
            const feedbackElement = useMemo(() => {
                if (typeof Audio === 'undefined') return null;
                return new Audio('/audio/beep.mp3');
            }, []);

            useEffect(() => {
                if (!audioElement || !feedbackElement) return;
                actions.attachMedia(audioElement);
                actions.attachFeedbackElement(feedbackElement);
                return () => {
                    actions.detachMedia(audioElement);
                    actions.detachFeedbackElement(feedbackElement);
                };
            }, [actions, audioElement, feedbackElement]);

            return (
                <div>
                    <div>Volume: {volume}</div>
                    <div>Stored: {storedVolume}</div>
                    <div>Muted: {String(muted)}</div>
                    <Slider
                        value={muted ? storedVolume : volume}
                        scaleMin={0}
                        scaleMax={1}
                        step={0.05}
                        width={'100%'}
                        onChange={(value) => actions.setVolume(value, { playFeedback: false })}
                    />
                    <button type="button" onClick={() => (muted ? actions.unmute() : actions.mute())}>
                        {muted ? 'Unmute' : 'Mute'}
                    </button>
                    <button type="button" onClick={() => void actions.playFeedback()}>
                        Play Feedback
                    </button>
                    <div>Attached Media: {audioElement ? 1 : 0}</div>
                </div>
            );
        }
        ```
