        # VideoStore
        Global video orchestration helpers built with Zustand. The store tracks one active video descriptor at a time and is intended to work with `VideoController`.

        ## React exports (reactive)
        React exports are reactive subscriptions for components updating automatically as state changes.

        ## Atomic selectors in React
        Use focused selectors so components only re-render when their selected slice changes.

        <RenderSource language={"tsx"}>
        ```
        import { Button, Video } from '@apple-pie/slice';
        import { useVideo, useVideoActions } from '@apple-pie/slice/stores/video';

        function VideoLauncher() {
            const activeVideo = useVideo();
            const actions = useVideoActions();

            return (
                <Button
                    onClick={() =>
                        actions.show({
                            component: Video,
                            props: {
                                src: 'https://example.com/video.mp4',
                                controls: 'simple',
                                muted: false,
                            },
                        })
                    }
                >
                    {activeVideo ? 'Video Open' : 'Open Video'}
                </Button>
            );
        }
        ```
        </RenderSource>

        ## Imperative usage outside React
        These APIs are useful for non-component event handlers and services.

        <RenderSource language={"tsx"}>
        ```
        import { Video } from '@apple-pie/slice';
        import { videoActions, getVideo } from '@apple-pie/slice/stores/video';

        export function openIntroVideo() {
            videoActions.show({
                component: Video,
                props: {
                    src: 'https://example.com/video.mp4',
                    controls: 'simple',
                    muted: false,
                },
            });
        }

        export function dismissActiveVideo() {
            if (getVideo()) videoActions.clear();
        }
        ```
